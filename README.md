# 520-FurnitureRecycleApp

A community-driven web platform for sharing, discovering, and recycling second-hand furniture.

## Overview

FurnitureRecycleApp is designed to help people reduce waste by making it easier to give away, find, and reclaim second-hand furniture within local communities. The platform allows users to post unwanted furniture, browse items nearby, and coordinate pickups with others in an organized and efficient way.

Unlike informal social media posts, this system provides a structured workflow for furniture sharing. Users can browse listings on a map, filter by category and location, request owner-listed items, update the status of street-found items, and earn points for contributing to the platform.

By supporting both reuse and community participation, the application promotes sustainability while reducing wasted trips and unnecessary disposal of usable furniture.

## Features

- **User Registration and Authentication**  
  Users can sign up with email and password or log in through a third-party provider such as Google.

- **Furniture Item Posting**  
  Users can post items they own or report furniture found on the street, including photos, descriptions, categories, and locations.

- **Interactive Map Discovery**  
  Nearby listings are shown on a map with location pins so users can quickly explore available items.

- **Search and Filtering**  
  Users can filter items by category, location, and item type to find relevant listings more efficiently.

- **Item Request System**  
  Users can send requests for owner-posted items and include a message to the item owner.

- **Reservation and Pickup Coordination**  
  Owners can review requests, reserve items for selected users, and coordinate pickup.

- **Pickup Confirmation**  
  After a successful transfer, users can confirm the pickup and remove the item from active listings.

- **Street Item Status Updates**  
  Users can verify whether a street-found item is still present, has been taken, or is no longer available.

- **Wishlist and Item Suggestions**  
  Users can save desired item categories and receive helpful suggestions when relevant listings appear.

- **Gamification and Community Points**  
  The platform rewards helpful user actions with points and can display a leaderboard of top contributors.

## Project Goal

The goal of this project is to build a practical and community-oriented platform that makes furniture reuse easier, reduces waste, and encourages sustainable habits through technology.
